# Flask Notes App

A simple full-stack Notes application built using Flask, HTML, CSS, and JavaScript.

## Features
- Add notes
- View notes
- Delete notes
- Clean and responsive UI

## Tech Stack
- Backend: Flask
- Frontend: HTML, CSS, JavaScript

## Project Structure

```
notes_app
│
├── app.py
├── requirements.txt
│
├── templates
│     └── index.html
│
└── static
      ├── script.js
      └── style.css
```

## Installation

1. Clone the repository

```
git clone <repo-link>
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```

4. Open browser

```
http://127.0.0.1:5000
```
